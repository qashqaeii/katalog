راه‌اندازی اولیه V2Bot

برای اجرای ربات روی سرور، مراحل زیر را دنبال کنید

1️⃣ دریافت سورس کد (با لایسنس)

ابتدا فایل‌های پروژه را روی سرور خود دانلود کنید:

کلون کردن مخزن گیت‌هاب

git clone https://{your_license_key}@github.com/qashqaeii/v2bot.git

توضیحات:

  • دستور git clone برای دانلود سورس کد پروژه از گیت‌هاب استفاده می‌شود.
  • {your_license_key} را با کلید لایسنس اختصاصی خود جایگزین کنید.

ورود به پوشه پروژه

cd v2bot

توضیحات:

  • دستور cd v2bot شما را وارد پوشه v2bot می‌کند که شامل تمام فایل‌ها و اسکریپت‌های پروژه است.

2️⃣ آماده‌سازی اسکریپت نصب

قبل از اجرای اسکریپت نصب، باید آن را به فرمت صحیح یونیکس تبدیل کنید:

نصب dos2unix

apt install dos2unix -y

توضیحات:

  • این دستور بسته dos2unix را نصب می‌کند که برای تبدیل فایل‌های متنی به فرمت یونیکس استفاده می‌شود.

تبدیل فایل install.sh به فرمت یونیکس

dos2unix install.sh

توضیحات:

  • این دستور فایل install.sh را به فرمت یونیکس تبدیل می‌کند تا از خطاهای مربوط به پایان خط (CRLF) جلوگیری شود.

اعطای دسترسی اجرا به فایل install.sh

chmod +x install.sh

توضیحات:

  • این دستور مجوز اجرایی را به فایل install.sh اضافه می‌کند تا بتوانید آن را مستقیماً اجرا کنید.

3️⃣ اجرای اسکریپت نصب 

./install.sh

توضیحات:

  • این دستور اسکریپت نصب را اجرا می‌کند که به‌صورت خودکار تمام بسته‌های مورد نیاز را نصب، پایگاه داده را راه‌اندازی و ربات را پیکربندی می‌کند.

4️⃣ اجرای ربات

از دستورات زیر برای کنترل ربات استفاده کنید:

شروع ربات

systemctl start v2bot

توضیحات:

  • این دستور با استفاده از systemd، سرویس ربات را راه‌اندازی می‌کند.

توقف ربات

systemctl stop v2bot

توضیحات:

  • این دستور سرویس ربات را متوقف می‌کند.

بررسی وضعیت ربات

systemctl status v2bot

توضیحات:

  • این دستور وضعیت فعلی سرویس ربات را نمایش می‌دهد و برای اطمینان از اجرای صحیح یا عیب‌یابی مشکلات مفید است.

مشاهده لاگ‌های زنده

journalctl -u v2bot.service -f

توضیحات:

  • این دستور لاگ‌های زنده سرویس ربات را نمایش می‌دهد که برای نظارت و عیب‌یابی فعالیت ربات مفید است.

⚙️ تغییر فایل config.py 

sudo nano ~/v2bot/config.py

توضیحات:

  • این دستور فایل config.py را در ویرایشگر متنی nano باز می‌کند.
  • در این فایل می‌توانید تنظیماتی مانند توکن ربات، اطلاعات پایگاه داده، رمزهای عبور و سایر پارامترها را تغییر دهید.
