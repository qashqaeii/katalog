راه‌اندازی اولیه V2Bot

راهنمای گام به گام نصب و راه‌اندازی ربات مدیریت سرویس‌های V2RAY

1

کلون کردن مخزن (با کلید لایسنس)

دانلود فایل‌های پروژه به سرور شما

کلون کردن مخزن گیت‌هاب

git clone https://{your_license_key}@github.com/qashqaeii/v2bot.git

جایگزین کردن {your_license_key} با کلید لایسنس اختصاصی شما

ورود به دایرکتوری پروژه

cd v2bot

وارد شدن به پوشه v2bot که حاوی تمام فایل‌ها و اسکریپت‌های پروژه است

2

آماده‌سازی اسکریپت نصب

تبدیل اسکریپت نصب به فرمت صحیح UNIX

نصب dos2unix

apt install dos2unix -y

نصب ابزار dos2unix برای تبدیل فایل‌های متنی به فرمت UNIX

تبدیل فرمت و اجازه اجرا

dos2unix install.sh
chmod +x install.sh

تبدیل فایل install.sh به فرمت UNIX و اعطای مجوز اجرا

3

اجرای اسکریپت نصب

نصب خودکار تمام وابستگی‌ها و پیکربندی ربات

اجرای اسکریپت

./install.sh

این اسکریپت به صورت خودکار تمام بسته‌های مورد نیاز را نصب و پیکربندی می‌کند

4

راه‌اندازی و کنترل ربات

دستورات مدیریت سرویس ربات

مدیریت سرویس

systemctl start v2bot

شروع سرویس ربات

systemctl stop v2bot

توقف سرویس ربات

systemctl status v2bot

مشاهده وضعیت سرویس ربات

journalctl -u v2bot.service -f

مشاهده لاگ‌های زنده سرویس ربات

5

پیکربندی ربات

تنظیم پارامترهای ربات در فایل پیکربندی

ویرایش فایل پیکربندی

sudo nano ~/v2bot/config.py

تنظیمات قابل ویرایش در فایل config.py:

  • توکن ربات تلگرام
  • اطلاعات اتصال به پایگاه داده
  • رمزهای عبور و دسترسی‌ها
  • سایر پارامترهای پیکربندی

نیاز به راهنمایی بیشتر دارید؟

برای اطلاعات بیشتر و راهنمایی تکمیلی، مستندات ما را مطالعه کنید یا با تیم پشتیبانی تماس بگیرید
