راه‌اندازی اولیه V2Bot

راهنمای گام به گام نصب و راه‌اندازی ربات مدیریت سرویس‌های V2RAY

1

آماده‌سازی سرور

به‌روزرسانی سیستم

sudo apt update && sudo apt upgrade -y

اطمینان از به‌روز بودن سیستم‌عامل و بسته‌های نصب شده

نصب پیش‌نیازها

sudo apt install python3 python3-pip docker.io -y

نصب Python، pip و Docker مورد نیاز برای اجرای ربات

2

دریافت و نصب ربات

کلون کردن مخزن

git clone https://github.com/v2bot/v2bot.git
cd v2bot

دریافت آخرین نسخه کد ربات از مخزن گیت‌هاب

نصب وابستگی‌ها

pip3 install -r requirements.txt

نصب تمامی کتابخانه‌های مورد نیاز پایتون

3

پیکربندی ربات

تنظیم فایل پیکربندی

cp config.example.yml config.yml
nano config.yml

تنظیمات ضروری در فایل پیکربندی:

  • توکن ربات تلگرام
  • اطلاعات اتصال به پنل سنایی
  • تنظیمات پایگاه داده
  • آدرس‌های API و Webhook
4

راه‌اندازی ربات

اجرای ربات

python3 bot.py

پس از اجرای موفق، ربات آماده دریافت دستورات خواهد بود

تست عملکرد

برای تست عملکرد ربات:

  • به ربات در تلگرام پیام /start ارسال کنید
  • از منوی اصلی، گزینه "وضعیت سیستم" را انتخاب کنید
  • اطمینان حاصل کنید که تمام سرویس‌ها در حال اجرا هستند

نیاز به راهنمایی بیشتر دارید؟

برای اطلاعات بیشتر و راهنمایی تکمیلی، مستندات ما را مطالعه کنید یا با تیم پشتیبانی تماس بگیرید
